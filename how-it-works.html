The suite

Three tools.
One production.

Each tool is a fully standalone app. Together, they form a continuous pipeline from first word to final call sheet.

01 · Write
WroteNote
Write · Break Down

Full screenplay editor with inline breakdown tools. Write your script, then annotate every element directly in the text.

  • Screenplay format editor
  • Scene-by-scene breakdown
  • Tag props, costumes, locations
  • Shot annotations with lens & movement
  • Character extraction from dialogue
  • Export to MediumShot or 80AD
02 · Prep
MediumShot
Shot List · Visual Prep

Shot listing and visual preparation. Import your script breakdown and build detailed shot lists with framing, lens, and equipment data.

  • Shot list builder grouped by scene
  • Auto-populated from WroteNote annotations
  • Framing, lens, movement per shot
  • Moodboard & reference imagery
  • Script viewer alongside shot list
  • Export shot list to 80AD schedule
03 · Produce
80AD
Schedule · Manage · Produce

Full production management. Shooting schedules, call sheets, budget, costume, phonebook — everything a 1st AD needs from prep to wrap.

  • Shooting schedule pre-filled from script
  • Call sheet builder
  • Props & costume tracking
  • Budget with breakdown line items
  • Crew phonebook (cast auto-imported)
  • Receives data from both WroteNote & MediumShot
Step by step

The workflow.

A single continuous pipeline. Every tool hands off exactly what the next one needs.

01 · WroteNote

Write the script.

Open WroteNote and write your screenplay using the full-featured script editor. Scenes, action, dialogue, transitions — standard industry format. When the draft is complete, switch to Breakdown mode.

Screenplay format Scene structure Character arcs
02 · WroteNote Breakdown

Break it down.

Select any text in the script and tag it. Highlight "the camera rig" → tag as Prop. Highlight "trench coat" → tag as Costume. Highlight a character entrance → mark it as a Shot. Each tag carries precise data: prop names, character associations, shot type, lens, movement.

Props tagged Costumes tagged Shots annotated Locations extracted Characters extracted
03 · Choose your path

Send to suite.

Click "Send to Suite" in WroteNote. A panel slides up asking: does this go to MediumShot first (for shot listing), or straight to 80AD? If it's your first time, you'll also be prompted to create or select a project. This becomes the shared project ID across all three apps.

New or existing project Path A → MediumShot Path B → 80AD direct
04 · MediumShot (Path A)

Build the shot list.

MediumShot opens with your shot annotations already imported — grouped by scene, with frame type, subject, lens, and movement pre-filled from your WroteNote breakdown. Add or edit shots, attach reference images, build your moodboard. When ready, send to 80AD.

Shot list auto-populated Scene grouping Frame · Lens · Movement Moodboard
05 · 80AD

Production ready.

80AD opens with your production fully scaffolded. The shooting schedule has one row per scene, pre-filled with actors and props. The costume department has every tagged prop and costume. The budget has a /bin/sh line item for every prop ready to price. The phonebook has every character as a cast stub. All you do is fill in the blanks.

Shooting schedule seeded Props & costumes imported Budget line items created Cast phonebook populated Call sheets
Under the hood

What gets transferred.

Every handoff is precise. Here's exactly what data moves between each tool.

WroteNote → MediumShot
Script → Shot List
Shot annotations
Scene, setup, shot number, frame, subject, lens, movement, notes
Script reference
Title, scene count, scene headings for cross-reference
Characters
All cast members extracted from dialogue blocks
WroteNote → 80AD (direct)
Breakdown → Production
Shooting schedule
One row per scene, with actors and props pre-filled
Props & costumes
Every tagged prop → Props tab. Every costume → Costume tab with character link
Budget + cast
/bin/sh line items per prop (dept: Props). Characters → phonebook as cast stubs
MediumShot → 80AD
Shot List → Schedule
Shots → schedule rows
Shot list grouped by scene becomes shooting schedule rows in 80AD. Shot type, subject, takes count, scene number all carried over.
